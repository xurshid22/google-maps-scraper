# 🗺️ Google Maps Scraper

> **100% Free** | No API Key needed | Extract business data from Google Maps

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/xurshid22/google-maps-scraper/blob/main/google_maps_scraper.ipynb)
[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

## Features
- Search any business in any city/country
- Extract website, phone, address, rating, reviews
- Export to CSV and JSON
- Works on Google Colab (no install needed!)

## Quick Start

### Google Colab (Recommended)
Click the badge above

### Local
```bash
pip install playwright
python -m playwright install chromium
python google_maps_scraper.py "restaurants Moscow"
```

## Output Fields
name, website, phone, address, rating, reviews, category, google_maps_url
